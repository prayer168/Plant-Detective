# 內容查核與圖像來源

查核日期：2026-07-19（Asia/Taipei）

## 主要來源

1. [臺灣物種名錄 TaiCOL](https://taicol.tw/) — 查核台灣物種接受名、學名與分類的主要入口。
2. [臺灣生命大百科 TaiEOL](https://taieol.tw/) — 查核植物形態、分布、果實與生態；例如榕樹、臺灣欒樹、小葉欖仁等條目。
3. [國立臺灣大學科學 Online 植物圖鑑](https://case.ntu.edu.tw/highscope/%E7%94%9F%E7%89%A9%E5%9C%96%E9%91%91/index.html) — 交叉核對科別與常見植物形態。
4. [農業部農業知識入口網：台灣原生植物資源之調查及蒐集—植物採集注意事項](https://kmweb.moa.gov.tw/knowledgebase.php?func=1&id=20937&type=13296) — 採集記錄、標籤與觀察順序。
5. [農業部農業知識入口網：種子小常識](https://kmweb.moa.gov.tw/subject/subject.php?id=26285) — 果實／種子分辨與地面撿拾方式。
6. [農業部農業 E 報：校園常見的有毒植物](https://epost.moa.gov.tw/theme_data.php?id=463&sub_theme=photo&theme=epost) — 金露花、馬利筋等校園植物安全提醒。
7. [彰化縣衛生局：不明野菜不要採食](https://www.chshb.gov.tw/node/190924853) — 不明植物不得採食與誤食風險。

## 教學簡化

- 圖版為 IMAGE 2.0 生成的比較示意，不代表同一株植物會在同一時間同時具有葉、花與成熟果實。
- 圖中標本按欄位排列，名稱與辨識重點以 HTML 文字為準；實地鑑定應合併葉序、葉形、花、果、樹皮與生長型等多項證據。
- 「常見」受地區、校園植栽歷史、海拔與管理方式影響，教材不宣稱每一所台灣校園都能看到全部 40 種。
